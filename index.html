🦠Neurons: Evoltionary Spiking Neural Network {2018-2019}

This was my first attempt at AI. And of course I didn't want to just learn TensorFlow. No, I wanted to make a spiking neural network from scratch. Thounsands of lines of code later, I had self assembling clusters of spiking neurons evolving using sexual reproduction.
It was object oriented using multitheading in Java and performance was good, yet it was held back by the cluster structure. I don't see a use in continuing unless I were to create visual tool for people to learn about creating and training neural networks. Unless I recieve funding for such a program, it's not going to happen.
One last interesting thing. At one point they found a flaw in my culling algorithm and started evolving to consum all 24GB of my computer's memory with their massive population. Evoltion always finds a way.
